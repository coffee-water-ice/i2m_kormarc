# core/name_data/

245 폴더에서 이관한 이름판별 데이터 자산. 700/710 필드(개인명 부출/기관명 부출)의
지시기호(성 유무, 개인명 vs 단체명) 판별과 900(원저자 한글명) 생성에 쓰인다.
`core/fields/marc_500_700_710.py`가 이 패키지를 사용한다.

## 구성 (원본: `245/245/`)

| 파일 | 역할 |
|---|---|
| `korean_surnames.py` | 한국 성씨 사전(frozenset, 길이별 인덱스) — 700 두 번째 지시기호 판별 |
| `korean_given_names.py` | 출생신고 통계 기반 이름 음절 판별 — 필명/실명 구분 |
| `korean_real_name_allowlist.py` | 2글자 필명 중 실명 취급할 예외 목록(한강/백석/이상 등) |
| `japanese_surnames.py` | 일본 성씨 한글 표기 사전 — 국적 판별 보조 신호 |
| `data/korean_given_name_weights.tsv` | 이름 음절 빈도 원본 데이터(대법원 전자가족관계등록 기반, 2008~2019) |

041/260/300/653 등 다른 필드 모듈은 이 데이터에 의존하지 않으므로 순환 참조 위험이 없다.
